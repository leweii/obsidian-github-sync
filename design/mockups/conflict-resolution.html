Conflict Resolution — 5 State Variants

High-fidelity HTML mockup of ConflictModal Page 6 from the design plan. Colors use Obsidian's CSS variable names so styles port 1:1 into the plugin's styles.css. Use this as the visual blueprint when redrawing in Figma — or as the direct reference when rewriting src/ui/ConflictModal.ts to replace the current ours/theirs/skip buttons.

  1. Idle — default three-pane diff with AI suggestion visible
  2. AI-Thinking — AI pane shows skeleton + spinner, "Take AI" disabled
  3. AI-Failed — AI pane replaced with error card + retry / switch provider
  4. Manual Edit — AI pane becomes inline textarea editor
  5. All Resolved — summary state, commit-ready
State 1

Idle — default three-pane diff

Local and Remote panes are dimmed to 55% opacity as reference material; the AI Suggestion pane is full-saturation with a soft accent tint. AI's picks are marked with ★ on only the lines where AI actually chose — not on every line — so attention compresses to 2-3 spots. Five hunk-level actions; file-level "Save & next" is independent at the bottom right. Confidence indicator is the gating mechanism for the bottom-left "Resolve all with AI" bulk action.

State 3

AI-Failed — provider error

The third pane is replaced by an error card with three explicit recoveries: Retry (same provider), Switch provider (open inline dropdown of configured providers — DeepSeek / Gemini / OpenAI), and Resolve manually (give up on AI for this hunk, fall through to Local/Remote/Edit). The reasoning panel mirrors the failure with the raw error message so users can paste it into the docs when reporting. Local and Remote remain fully usable — the failure doesn't block the user.

State 4

Manual Edit — inline textarea

Clicking "Edit manually" replaces the AI pane with a focused textarea pre-filled with the AI suggestion (or, if AI failed, with Local content as the starting point). The pane border glows with accent color to indicate active focus. Hunk actions collapse to Save edit / Cancel. Keyboard shortcuts shown in the hint strip below the editor (⌘+Enter to save, Esc to revert). Local/Remote panes stay visible so the user can copy-paste fragments while editing.
