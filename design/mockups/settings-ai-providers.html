Settings — AI Providers & Privacy

Page 2 of the design plan. This is where users configure DeepSeek / Gemini / OpenAI tokens, decide a fallback order when one provider rate-limits or fails, control what gets sent to AI (privacy), and watch their monthly token budget. Built as a settings tab inside the existing GitHubSyncSettingTab, not a separate modal.

  1. Main configured state — 3 providers (2 active + 1 disabled), DeepSeek expanded, fallback chain visible, full Privacy + Usage
  2. Empty state — no providers yet, single CTA to add one
  3. Limit-warning callout — banner that inserts at top of Usage when 80% of soft limit is hit
State 1

Configured — 2 active providers + fallback chain

Four sections, each with an iconic header that mirrors the existing wizard's hero style. AI Providers shows three cards: DeepSeek (expanded inline edit), Gemini (collapsed, fallback role), OpenAI (disabled). Fallback chain is a draggable pill list — the active provider tries first, each step is fallback for the previous. AI Privacy has a master kill-switch, three granular toggles, an exclude-from-AI textarea, and a per-repo overrides table. AI Usage shows monthly token spend across providers as a stacked bar with soft/hard cost limits.

AI Privacy

Control what file contents and metadata are sent to AI providers. Defaults are conservative — only conflicting hunks plus minimum surrounding context.

Enable AI conflict resolution
Master switch. When off, the conflict modal shows only Local / Remote / Manual options (no AI Suggestion pane).
Send file paths to AI
Lets AI use filenames and folder structure to make better merge decisions (e.g. notes/Project Alpha.md). Without this, AI only sees raw text.
Send git metadata
Includes commit messages, author, and timestamps in the AI prompt. Helps disambiguate which version is newer.
Send surrounding context lines
Adds ±10 lines around each hunk so AI has document context. Increases token spend by ~30% per conflict.
Glob patterns. Conflicts in these paths skip AI entirely and fall through to manual resolution. Same syntax as the global Ignore Patterns setting.
